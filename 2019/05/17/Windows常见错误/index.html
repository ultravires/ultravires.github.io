Windows常见错误

此文章记录了本人在使用 Windows 操作系统中所遇到的一些问题，文章的篇幅会随着个人遇到的问题越来越长。

使用对象的权限被否定

原因：安装了office控件，但该控件没有权限。如：AxMath

解决办法：删除此控件或赋予该控件管理员权限

如何找：

  1. 打开注册表
  2. 计算机\HKEY_LOCAL_MACHINE\SOFTWARE\WOW6432Node\Microsoft\Office
  3. 找到你的office目录后点击响应的office产品查看其装有的控件

我这里PowerPoint下未装任何控件（AxMath已被我卸载）。如果你有，请自行寻找安装的控件位置进行赋权或卸载操作。
